Установка необходимых библиотек (инструкция предназначена для пользователей Windows!):
1. Открываем CMD от имени администратора (WIN -> вводим CMD --> Запустить от имени администратора)
2. Вводим следующую команду: cd Путь до проекта
3. Вводим последннюю команду: pip install -r requirements.txt
4. Библиотеки установлены!

Краткий экскурс по программе:
- Данная программа предназначеная для взаимодействия с таблицей базы данных.
- Существующие операции: удаление, добавление, редактирование, поиск, вывод всех составляющих базы данных.
- При открытии программы вам чётко дадут знать, чего от вас хотят. Следуйте указаниям, написанным в самой программе!
